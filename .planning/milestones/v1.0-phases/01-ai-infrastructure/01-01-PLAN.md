Create Key entities for secure API key storage using Drush:

```bash
cd html

# Create Deepseek API Key
drush eval "
\$key = \Drupal::entityTypeManager()->getStorage('key')->create([
  'id' => 'deepseek_api_key',
  'label' => 'Deepseek API Key',
  'description' => 'API key for Deepseek chat/generation',
  'key_type' => 'authentication',
  'key_provider' => 'config',
  'key_input' => 'text_field',
]);
\$key->setKeyValue(getenv('DEEPSEEK_API_KEY') ?: 'REPLACE_WITH_ACTUAL_KEY');
\$key->save();
echo 'Created Deepseek API key';
"

# Create OpenAI API Key

\$key = \Drupal::entityTypeManager()->getStorage('key')->create([
```
