Create a Search API server using the AI Search backend:

```bash
cd html

# Create the AI Knowledge Garden server via Drush
drush eval "
\$server = \Drupal::entityTypeManager()->getStorage('search_api_server')->create([
  'id' => 'ai_knowledge_garden',
  'name' => 'AI Knowledge Garden',
  'description' => 'Vector search using Milvus and OpenAI embeddings',
  'backend' => 'ai_search',
  'backend_config' => [
    'vdb_provider' => 'milvus',
    'embedding_provider' => 'openai',
    'embedding_model' => 'text-embedding-3-small',

]);

echo 'Created AI Knowledge Garden server';
"
```
