## Architecture

```
DeepChatApi::api() → AiAssistantApiRunner::process() → AgentRunner::runAsAgent()
  → AiAgentEntityWrapper::determineSolvability() [the agent loop]
    → $provider->chat($input) [calls Gemini with tools]
    → if response has tools && loopedEnabled → recurse (execute tools, call LLM again)
    → if no tools → finished=TRUE, return
  → AgentRunner gets last ChatMessage from history
  → DeepChatApi::createResponse() checks getTools() → sets should_continue
```
