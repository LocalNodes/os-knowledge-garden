       ```
       mcp__coolify__deployment({ action: "get", uuid: "<deployment-uuid>", lines: 100 })
       ```

    - Docker needs to build the opensocial image (installing PHP extensions)

    - Then enables localnodes_platform and demo content (1-2 minutes)
    - Then indexes content in Solr (1-2 minutes)
    - Then runs cron for vector indexing (1-2 minutes)
