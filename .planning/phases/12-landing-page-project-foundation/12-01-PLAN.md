    **Step 1: Initialize project.** Run `npx nuxi@latest init ../localnodes-onboarding` to scaffold. If the CLI prompts for options, accept defaults. Then `cd ../localnodes-onboarding` and add modules: `npx nuxi module add ui` and `npx nuxi module add content`. Run `npm install`.

    **Step 2: Configure nuxt.config.ts.** Set up modules (`@nuxt/ui`, `@nuxt/content`), CSS (`~/assets/css/main.css`), dark mode preference (`colorMode.preference: 'dark'`), prerender rule for `/` (`routeRules: { '/': { prerender: true } }`), HTML lang attribute, favicon link, and `compatibilityDate: '2025-07-15'`. Do NOT set the Vercel preset manually -- it auto-detects. Do NOT install or reference `@nuxt/ui-pro` (legacy v3 package).

    **Step 3: Configure app.config.ts.** Set Nuxt UI colors: `primary: 'teal'`, `secondary: 'amber'`, `neutral: 'zinc'`.

    **Step 4: Create main.css.** Import Tailwind and Nuxt UI with `@import "tailwindcss"` and `@import "@nuxt/ui"`. Add the dark custom variant: `@custom-variant dark (&:where(.dark, .dark *))`. Add root CSS variables for primary/secondary and dark mode overrides for bg colors. Add `scroll-behavior: smooth` to `html` for section navigation.

    **Step 5: Create app.vue.** Wrap with `<UApp>` containing `<NuxtLayout>` and `<NuxtPage />` inside it. This is critical -- UApp is required for toasts, tooltips, and overlays to function.

    **Step 6: Create error.vue.** Simple error page using `<UPageHero>` showing error status and message with a "Go Home" button linking to `/`.

    **Step 7: Create default.vue layout.** Flex column layout (`min-h-screen flex flex-col`) with `<AppHeader />`, `<main class="flex-1"><slot /></main>`, and `<AppFooter />`.

    **Step 8: Create AppHeader.vue.** Use `<UHeader>` component with:
    - Left slot: NuxtLink to `/` with "LocalNodes" text logo (font-bold text-xl)
    - Center slot: `<UNavigationMenu>` with anchor links to page sections: How it Works (#how-it-works), Features (#features), Pricing (#pricing), Communities (#communities)
    - Right slot: `<UButton>` "Get Started" linking to `/onboarding` with `color="primary"`

    **Step 9: Create AppFooter.vue.** Use `<UFooter>` or a simple footer with:
    - Left: "LocalNodes" brand + copyright
    - Center/right: Links to sections, link to cascadia.localnodes.xyz as "Live Demo"
    - Bottom: "Built for communities, by communities" tagline

    **Step 10: Create content.config.ts.** Configure the content module with default collection pointing to `content/` directory.

    All Vue code lives under `app/` directory (Nuxt 4 convention). Config files at project root. `server/` directory left empty for future BFF routes.
